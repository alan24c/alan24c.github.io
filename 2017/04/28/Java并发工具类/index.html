线程的创建

  • 线程可以分为两个方面：线程运行的任务(task) 和 运行线程的实体(thread)
    • 线程的任务实体 task
      • public void Runnable::run()
      • public void Thread::run()
    • 线程的构造
      • 使用 Runnable 对象(提供 task)初始化 Thread
      • 直接继承 Thread 类,并且覆盖 run() 方法(提供 task)
    • 线程的运行
      • public void Thread::start()
  • 线程池框架
    • interface:: Executor
      • public void Executor::execute(Runnable) 运行线程
    • 构造线程池
      • Executor.newFixedThreadPool(threadNums) -> 构造固定数目的线程，每次提交一个任务,创建一个线程,直到达到最大的线程数目
      • Executor.newCachedThreadPool -> 无限大小的线程池,会回收空闲线程
      • Executor.newSingelThreadExecutor -> 单线程数量的线程池
      • Executor.newScheduledThreadPool -> 固定长度的线程池，并且可以定时调度来执行任务
    • 自定义线程池的执行方式
      • 覆写 Executor::execute(Runnable)方法
    • interface::ExecutorService
      • 继承 Executor，添加了一些关于 Executor 生命周期的管理方法
      • 生命周期:运行,关闭(不再接受新的任务提交),已终止(已经提交的任务已经全部运行)
      • RejectedExecutionException 异常
